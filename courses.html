Brief Description of Each Course:

CS101 – Introduction to Programming:

A foundational course that introduces students to programming using Python, focusing on problem-solving and basic algorithms.

CS205 – Data Structures & Algorithms:

Covers fundamental data structures such as lists, arrays, and trees, along with algorithm analysis for optimizing program performance.

CS310 – Machine Learning Fundamentals:

An introduction to machine learning, covering essential algorithms such as linear regression, neural networks, and AI applications in data analysis.

CS420 – Secure Software Development:

Focuses on techniques for developing secure software, including code analysis for vulnerabilities and modern encryption methods.

CS515 – Cloud Computing & Security:

Explores cloud computing infrastructure, data security in cloud environments, and hands-on applications with AWS and Google Cloud.
